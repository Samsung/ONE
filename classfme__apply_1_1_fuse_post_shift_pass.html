Detailed Description

Pass to fuse CircleCustom(PostShift) to preceding Ops.

BEFORE 

        [Node1]
          |
         [Op]
        /    \

[PostShift] [Node2]

AFTER 

  [Node1]
   /  \

[Op'] [Op] | [Node2]

NOTE Op' is clone of Op with updated bias.
