Detailed Description

Tensor object that indirects to the tensor it is pointing to.

A model I/O tensor could be two types.

  1. UserTensor, if it is the primary graph
  2. Any other derivative of IPortableTensor from another backend, otherwise

To support these, this object indirects everything to the actual tensor pointer. Exceptionally if it is UserTensor, this class creates and manages it.
