Add an extended operation to a model.

Parameters
[in]modelThe model to be modified.
[in]typeThe type of extended operation.
[in]inputCountThe number of entries in the inputs array.
[in]inputsAn array of indexes identifying each operand.
[in]outputCountThe number of entries in the outputs array.
[in]outputsAn array of indexes identifying each operand.
Note
The operands specified by inputs and outputs must have been previously added by calls to ANeuralNetworksModel_addOperand.
Attempting to modify a model once ANeuralNetworksModel_finish has been called will return an error.
See ANeuralNetworksModel for information on multithreaded usage.
Returns
ANEURALNETWORKS_NO_ERROR if successful.
